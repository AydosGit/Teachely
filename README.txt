INSTRUCTIONS: HOST TEACHELY FOR FREE

1. FRONTEND (STATIC WEBSITE)
-----------------------------
Option: Use Vercel, Netlify, or Firebase Hosting

Steps for Vercel:
- Go to https://vercel.com/
- Create account, create new project
- Upload contents of 'frontend' folder
- Set backend URL in 'script.js'

2. BACKEND (API SERVER)
------------------------
Option: Use Railway.app or Render.com (Free)

Steps for Railway:
- Go to https://railway.app/
- Create account
- Start a new Node.js project
- Upload 'backend' folder contents
- Add a .env file with your MongoDB URI (see .env.example)
- Deploy and get the backend URL
- Paste backend URL into frontend/script.js

3. DATABASE
-----------
Use MongoDB Atlas (https://www.mongodb.com/cloud/atlas)
- Create free account
- Create a database named 'teachely'
- Create a user and get the connection URI
- Use it in your backend .env file

Enjoy building with Teachely!
